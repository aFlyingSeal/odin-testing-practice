## Odin Testing Practice

This repo showcases the usage of **Jest**, a JavaScript framework for testing applications, as part of The Odin Project curriculum for future projects that requires test-driven development.

Have a good day reading this.

---

## Key Features:

- Create your own JavaScript source code and write test suites for it.
- Usage of **Jest** for test-driven development.

## Built With:

JavaScript (ES Modules) and Jest

## Run This Project Locally:

1. Clone this repository:

```bash
    git clone https://github.com/aFlyingSeal/odin-testing-practice.git
    cd odin-testing-practice
```

2. Install dependencies:

```bash
    npm install
```

3. Run a specific test suite:

```bash
    npm test -- .\test\file-name.test.js
```

4. Run every single test suites:

```bash
    npm test
```
